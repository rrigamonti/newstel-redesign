# Newstel.com Design Analysis

## Color Scheme (Extracted from Screenshot)
- **Primary Blue/Navy**: #4A5F7F (dark blue-grey from header/hero)
- **Accent Coral/Red**: #FF6B6B / #E85D5D (from "Multilingual" text and CTA buttons)
- **Light Grey**: #F5F5F5 / #E8E8E8 (background sections)
- **White**: #FFFFFF (text on dark backgrounds, section backgrounds)
- **Dark Text**: #2C3E50 or similar dark grey/navy

## Typography
- Clean, modern sans-serif fonts
- Bold headlines with good hierarchy
- Mix of regular and bold weights

## Key Content Sections
1. Hero section with main value proposition
2. Client logos carousel
3. Services overview (24/7 Support, BPO)
4. About section with stats
5. Values section
6. Locations/offices
7. Technology integration
8. Careers
9. Contact form

## Design Elements to Modernize
- Curved section dividers
- Card-based layouts
- Statistics display
- Location cards
- Icon usage
- Spacing and white space
- Modern gradients and shadows
